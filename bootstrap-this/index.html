All About Bootstrap!

A brief guide you need to read before your first bootstrap encounter.

Bootstrap may look quite confusing in the beginning, but it gets easier once you get a hang of this amazing developing tool. To understand Bootstrap, one might need to have a basic understanding of HTML and CSS (of course) and all the basic elements that come with those languages.

Background

Bootstrap was originally named “Twitter blueprint.” In the beginning, it was just made as a framework to endorse and support consistency across internal tools. Before Bootstrap, there were several libraries that were being used for the development of interfaces. A large number of libraries caused a lot of inconsistencies. Along with the actual developer, Mark Otto and a few colleagues (several other team members from Twitter) contributed to this project in the Twitter "Hack Week." Bootstrap was then introduced as an open source project. It had a twelve-column responsive grid layout system. It was no longer used solely as an internal tool maintainer for Twitter.

Why is Bootstrap used at a scale it is being used today?

It’s simple, fun and easy to understand! Bootstrap as an interface developing tool can do a lot of things. Here are some of the features that make Bootstrap a widely used tool:

1. It is beginner friendly

Anyone, literally anyone can use Bootstrap (of course a basic understanding of HTML and CSS is required), but as a tool that comes with pre-sets and components ready to be used, it doesn’t take much time to get started, which is probably the first reason why it is appealing.


2. Easily Customizable

Even though Bootstrap is designed in responsive 12-column grids, layouts, and components, it is also very easy to customize. Whether you need a fixed grid or a responsive one, it can be made possible by making a few changes. Offsetting and nesting of columns are also easy to do in both CPU-based and mobile-based browser grids.


3. Responsiveness

Bootstrap is a responsive framework that incorporates a fluid 12 grid system for creating mobile ready websites. It uses two patterns depending on what the project demands and deems better: Firstly, we have the grid system, where one can use grids by laying down the 12 column width metrics for the website layout. Secondly, we have the flex system which allows faster and easier management of individual Bootstrap components like size of columns in the grid or the layout of content within a grid.

Bootstrap also has Responsive Breakpoints. These breakpoints aid in building complex responsive layouts which are more straightforward due to the presence of five sets of classes. The five classes cover extra-large, large, medium, small, and extra-small screen widths.

Bootstrap also has Sass variables and mixins, and dozens of predefined classes.

Borders

Border utilities help in applying borders/border radius to elements. These are a great way to style buttons and images.

Borders can be of two types: Additive and Subtractive

border border border border border

Subtractive border utilities helps one from removing borders from an element. You can achieve that by simply putting a 0 after the border-top or border-right value, like this: border-top-0


Positioning

These utilities help in determining the positioning of an element.

A few common values in positioning include:

  • position-static
  • position-relative
  • position-absolute
  • position-fixed
  • position-sticky

Although they are quick and easy to use, they are not responsive in nature.


Spacing

Spacing is an essential utility which consists of both padding and margin, to help enhance the appearance of your elements.

class="pl-4">Spacing is defined by two properties:

  • m stands for margin
  • p stands for padding

Next are the sides:

  • t - stands for margin-top or padding-top
  • b - stands for margin-bottom or padding-bottom
  • l - stands for margin-left or padding-left
  • r - stands for margin-right or padding-right
  • x - stands for both left & right
  • y - stands for both top & bottom
  • blankstands for setting a margin or padding on all 4 sides of the element

Sizing of these margins and padding depends on numeric values ranging from 0-5 and then auto.

There are additional utilities like Text, Display, Flex, Float, Overflow, Sizing and many others.

Drawback of Bootstrap

Everything has its pros and cons. We have talked about the features that make Bootstrap appealing. Now we will discuss a few flaws and drawbacks of Bootstrap. One of the biggest drawbacks is its free availability. Since it is a "Free for all" program, there are higher chances of people using common themes on their individual websites. Now, there is big competition to develop the most unique and different website using the same library that maybe 10,000+ people are already using or have used. Due to increased use in this modern day and time where websites have such a big part of a person’s everyday life, it has become quite easy to determine which websites and applications have used the same template unless they have been heavily and massively customized.
